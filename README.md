# Setapp Hero Clone

This is a small Next.js + Tailwind project that reproduces the hero section and navbar from the screenshots.

Quick setup:

1. Place your downloaded images into `public/images/`.
   - Expected filenames (match the screenshots): `pics.svg`, `teamwork.svg`, `plan.svg`, `code.svg`, `setapp-icon-birthday-15.svg`, `apple.svg`, `google.svg`, `div.logo.svg`.
2. Install dependencies:

```bash
npm install
```

3. Run dev server:

```bash
npm run dev
```

Notes:
- The layout uses Tailwind CSS. Fine-tune exact spacing and font weights depending on your local assets.
- If you want pixel-perfect rendering, ensure the same SVGs/PNG assets from your screenshots are placed in `public/images`.
